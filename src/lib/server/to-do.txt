На сервере:
    - CRUD данных пользователя
    - CRUD организации
    - CRUD проектов
    - CRUD отделов
    Подсчёты:
        Общие:
            - Подсчёт задач выполненных / не выполненных
            - Смотрим дедлайны (если дедлайн есть и мы его просрочили - эффективность низкая,
                если дедлайн не пропущен, то эффективность - нормальная,
                если сильно раньше, то эффективность - высокая)
            - Подсчёт проектов по статусу
                - Не выполненные задачи по проекту (кол-во)
            - Проекты по владельцу
            - Число проектов с наибольшим выполненным кол-вом задач
        Проекта:
            - Подсчёт заданий присвоеных одному человеку
            - Смотрим дедлайны (если дедлайн есть и мы его просрочили - эффективность низкая,
                если дедлайн не пропущен, то эффективность - нормальная,
                если сильно раньше, то эффективность - высокая)
            - Общие задачи по исполнителю и статусы задач (просрочено, запланированно, выполнено и не запланированно)
            - Задачи по исполнителю
            - Предстоящие задачи по исполнителю
